##### split의 차이
- input에 123 넣는 경우
```python
a, b, c = map(int, input().split()
1 2 3
a, b, c = map(int, input())
123
```
  - split의 여부에 따라 띄어쓰기 차이가 생김.
  
  ##### print
```python
a = 'happy'
b = 'hacking'

print(a, end="@") #default 값은 '\n'
print(b)
#happy@hacking

print(a, b) sep = "\n" #default 값은 ','    
'''
happy
hacking
'''
print(a, b, c) = map(int, input().split())
>>>1 2 3
print(a, b, c, sep='&', end='!')
# 1&2&3!
```
