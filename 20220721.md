##### split의 차이
- input에 123 넣는 경우
```python
a, b, c = map(int, input().split()
1 2 3
a, b, c = map(int, input())
123
```
  - split의 여부에 따라 띄어쓰기 차이가 생김.
  
  ##### print

a = 'happy'
