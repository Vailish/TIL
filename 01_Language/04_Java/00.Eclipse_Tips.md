# 이클립스 팁

## 이클립스 단축키
- Eclipse 단축키 확인 및 변경 :
   - Eclipse의 모든 단축키는
   - Window >> Preferences >> General >> Keys

## Java Editor 단축키
- 단축키 리스트 보기 : Ctrl + Shift + L
- 저장 및 컴파일 : Ctrl + S
- 저장(열려진 모든 파일) 및 컴파일 : Ctrl + Shift + S
- 한 줄 복사 : Ctrl + Alt + up/down
- 한 줄 삭제 : Ctrl + d
- 한 줄 이동 : Alt + up/down
- 실행 : Ctrl + F11
