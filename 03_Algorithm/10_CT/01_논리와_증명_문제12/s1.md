n^3이 3의 배수이면 n은 3의 배수임을 증명하라.
```
명제의 대우는 아래와 같습니다.
n이 3의 배수가 아니면, n^3은 3의 배수가 아니다

1) n = 3k - 2 일 때, (k는 자연수)
    n^2 = (3k -2)^2
        = 9k^2 -12k + 4
        = 3(3k^2 - 4k +1) + 1
        3k + 1의 형태로 나오므로, n^2은 3의 배수가 아니다.

2) n = 3k - 1 일 때
    n^2 = (3k -1)^2
        = 9k^2 -6k + 1
        = 3(3k^2 -2k) + 1
        3k + 1의 형태가 나오므르, n^2은 3의 배수가 아니다.

1) 과 2)에 따라서 대우가 참이 되고, 대우가 참인 명제는 참이 되므로, n^3이 3의 배수이면 n은 3의 배수이다.
```
